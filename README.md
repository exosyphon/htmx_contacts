## HTMX Example with Ruby
Example Inline edit form using htmx

### Install Ruby 3.2.2
- `asdf plugin add ruby https://github.com/asdf-vm/asdf-ruby.git`
- `asdf install ruby 3.2.2`
- `gem install bundler`

### Install Dependencies
- `bundle install`
 
### Run
- `ruby server.rb`

visit `localhost:4567` in your browser
